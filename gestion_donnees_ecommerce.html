Optimisez de la gestion des données de la boutique¶

BottleNeck gre actuellement ses stocks via un EPR qui n'est pas li au site de vente en ligne. La gestion de ceux-ci est complexe et les analyses des ventes sur le Net manque de visibilit. Une table relationnelle entre les donnes EPR et Web a t tablie avec la liste exhaustive des produits.

Nous avons donc deux bases de donnes, l'une pour l'EPR et l'autre pour l'e-shop et une table relationnelle. Comment simplifier la gestion des stocks et amliorer la visibilit des ventes ?

Les tapes :

  • Rapprocher les 2 bases de donnes
  • Mise en vidence du chiffre d'affaire
  • Analyse des donnes de prix

def import_xls_to_dataframe(filename):
    """
    Convert xls file to pandas dataframe
        
    args
        filename: name of file with extension
    returns
        pandas dataframe
    """
    try:
        return pd.read_excel(DATA_PATH + filename, engine='openpyxl')
    except Exception as e:
        print(">>> Le fichier {} est introuvable.".format(filename))
        sys.exit(e)

def is_primary_key(dataframe, column):
    """
    Check if values of column are primary keys
    
    args:
        dataframe: pandas dataframe
        column: column name to check
    returns:
        boolean
    """
    if column not in dataframe.columns:
        raise ValueError("'{}' n'est pas une colonne de la dataframe".format(column))
    
    return len(dataframe.drop_duplicates(subset=column)) == len(dataframe)

Toutes les valeurs l'extrieur de la limite infrieure et suprieure sont des outlier.
